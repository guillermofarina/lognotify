Descripción detallada

Clase principal de la aplicación cliente de logNotify desde la que se controlan las funcionalidades de la misma. Una instancia de ClienteDeNotificaciones debe de ser inicializada mediante la función ClienteDeNotificaciones::inicializar() para empezar a funcionar
