Descripción detallada

Clase abstracta que define la estructura e interfaz de una condición de una Regla para el Filtro. Una Condicion se compone de una expresión regular en formato ECMAScript y una fución (virtual pura, implementada en las clases derivadas) evaluar() que decide si la condición se cumple o no para una notificación
